Marketing

Módulo

Objetivo: Implementar estrategias para dar a conocer la cafetería, atraer más clientes y destacar la marca usando campañas, redes sociales y promociones en medios físicos y digitales.

Fecha_fin Crear Leer Actualizar Eliminar (cuando pase)

Atributos • Para qué sirve

Atributo Para qué sirve
id_marketing Identifica cada promoción o campaña con un número único para evitar confusión.
nombre Ponerle un nombre sencillo y reconocible a la promoción (ej. “2x1 en café”).
objetivo Explicar qué se busca lograr con esa promoción (ej. vender más en la tarde).
medio Indicar dónde se anunciará (ej. cartel, redes sociales, volante).
fecha_inicio Saber el día exacto en que empieza la promoción.
fecha_fin Saber cuándo termina y así no tener promociones fuera de fecha.

Carta

Módulo

Objetivo: Crear y organizar un menú atractivo y fácil de entender que ayude a los clientes a elegir productos y aumente las ventas, disponible en formato físico y digital.

Tabla • Acciones por atributo (CRUD)

Crear Leer Actualizar Eliminar
Atributo Acciones
Id_producto Crear Leer
Nombre Crear Leer Actualizar
Tipo Crear Leer Actualizar
Precio Crear Leer Actualizar
Disponible Crear Leer Actualizar
Promoción Crear Leer Actualizar Eliminar (si ya no aplica)

Atributos • Para qué sirve

Atributo Para qué sirve
id_producto Identificar cada producto (bebida, comida, postre) con un número único para tener orden.
nombre Mostrar el nombre del producto en el menú para el cliente.
tipo Clasificar el producto (bebida, comida, postre).
precio Indicar el costo del producto para control interno y mostrar al cliente.
disponible Señalar si el producto está disponible o no (por ejemplo, si falta algún ingrediente).
promoción Indicar si el producto tiene alguna oferta especial o descuento.

Relación entre Marketing y Carta

Cómo se conectan los módulos
Atributo en Marketing Atributo en Carta Tipo de Relación Explicación
nombre promoción Relación directa Una promoción creada en Marketing se asocia a un producto específico de la carta (ej. “2x1 en Latte”).
objetivo tipo Relación estratégica Los objetivos de las campañas ayudan a decidir qué tipo de productos promocionar (bebidas, postres, etc.).
medio nombre Relación informativa Indica cómo y dónde se promocionará el producto seleccionado de la carta.
fecha_inicio / fecha_fin disponible Relación temporal La promoción solo aplica si el producto está disponible y dentro de las fechas indicadas.
